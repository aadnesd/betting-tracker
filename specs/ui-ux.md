# UI/UX Spec

## Core Screens
- Dashboard (summary + recent activity)
- Matched sets list with expandable details
- Matched set detail + reconciliation actions
- AI intake flow (upload → parse → review → save)
- Reporting and exposure

## UX Rules
- Server components by default; `"use client"` only for interactive pieces.
- Clear, audit‑friendly tables.
- “Quick Add” for common case.
- Minimal mobile layout for on‑the‑go logging.
- Emphasize transparency in calculations (show breakdowns).
